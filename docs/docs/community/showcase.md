# Showcase

Explore tools and resources built entirely in Jac for Jac.

## Jac Playground

**[playground.jaseci.org](https://playground.jaseci.org/cl/app){target=_blank}**

Write, run, and experiment with Jac code directly in your browser -- no installation required. The playground is the fastest way to try out language features, test snippets, and share examples with others.

## Jac GPT

**[jac-gpt.jaseci.org](https://jac-gpt.jaseci.org){target=_blank}**

An AI assistant trained on the Jac language and ecosystem. Ask questions about syntax, get help debugging code, or explore Jac concepts interactively.

## Jac Builder

**[jac-builder.jaseci.org](https://jac-builder.jaseci.org/){target=_blank}**

A visual builder for Jac applications. Design and build Jac projects with an interactive interface.
