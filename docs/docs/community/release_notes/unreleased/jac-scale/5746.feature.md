- **Feature: kvstore atomic compare-and-set primitives**: Added `set_nx_with_ttl(key, value, ttl)` (atomic `SET NX EX`) and `delete_if_equals(key, expected)` (atomic Lua compare-and-delete) to the Redis kvstore. Enables app-level distributed locks, single-winner debounces, and fence-token release patterns without opening a second Redis connection pool. Both methods are Redis-only and raise `NotImplementedError` on MongoDB.
